# 📊 Customer Insights Dashboard

A full-stack customer analytics dashboard that segments customers and predicts spending behavior using machine learning.

---

## 🚀 Live Demo
🔗 https://customer-insights-1604.streamlit.app

---

## 📌 Overview
This project analyzes customer data to:
- Segment customers into groups using clustering
- Predict spending behavior using regression
- Visualize insights through an interactive dashboard

---

## 🛠️ Tech Stack
- Python
- Streamlit (Frontend)
- Scikit-learn (Machine Learning)
- Pandas (Data Processing)

---

## ⚙️ Features
- 📄 View complete customer dataset  
- 📊 Customer segmentation using K-Means clustering  
- 📈 Spending prediction using Linear Regression  
- 🎯 Filter customers by segment  
- 📊 Interactive visualizations and insights  

---

## 🧠 Machine Learning Models
- **K-Means Clustering** → Customer segmentation  
- **Linear Regression** → Spending prediction
